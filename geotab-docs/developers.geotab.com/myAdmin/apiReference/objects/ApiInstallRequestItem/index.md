**Introduction**

Stores installation service request information for a single item in an order.

**Properties**

## InstallContactId

Id of the contact that will handle the installation.

## InstallLocationComments

User's comments about the location of the installation.

## ProductCode

The product code of the install service product for this install request item.

## VehicleAvailabilityWindowOption

Int selecting when the vehicle is available for the installation. 0 for Unset, 1 for Business Hours, 2 for After Hours, 3 for Weekends.
