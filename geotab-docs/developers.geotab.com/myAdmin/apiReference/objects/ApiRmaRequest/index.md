**Introduction**

RMA request details.

**Properties**

## Account

The[ApiAccount](../ApiAccount/index.md)that the RMA belongs to.

## CaseId

The ID of the support case related to the RMA request.

## CaseUrl

URL to the support case related to the RMA request.

## Comments

Comments related to the RMA request.

## Id

ID of the[ApiRmaRequest](index.md)record.

## Items

The list of[ApiRmaRequestItem](../ApiRmaRequestItem/index.md)that belong to the RMA request.

## RequestDate

Date and time the RMA was requested.

## ResellerReference

Reseller's reference code to the RMA.

## Rma

The associated RMA.

## RmaType

The[ApiRmaType](../ApiRmaType/index.md)of the RMA.

## User

The[ApiUser](../ApiUser/index.md)that belongs to the RMA.

## UserContactId

The ID of the[ApiUserContact](../ApiUserContact/index.md)for this RMA request.
