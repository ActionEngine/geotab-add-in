**Introduction**

The result returned from a[ApiDeviceInstallRequest](../ApiDeviceInstallRequest/index.md).

**Properties**

## Comments

Comments regarding the device.

## FirmwareVersion

Current firmware version running on the device.

## LastServerCommunication

Date and time device last communicated with the server.

## LastValidEngineRecord

Date and time of last valid engine record.

## LastValidGpsRecord

Date and time of last valid GPS record.

## PossibleIssues

Indicates any possible issues with the device.

## Request

The[ApiDeviceInstallRequest](../ApiDeviceInstallRequest/index.md)the result is based on.

## ResultDateUtc

The date and time of the result record.

## SimActive

Indicates whether the device's SIM card is active.

## SimNumber

Device's SIM card number.
