**Introduction**

Contains the most recent database and VIN for a device.

**Properties**

## CreatedDate

The date the record for this device, database and VIN was first created.

## DatabaseName

The name of the most recent database the device was assigned to.

## DeviceId

The ID of the device for this record.

## Id

Database Id of the device database.

## StatusDate

The date that this record was last confirmed.

## Vin

The most recent VIN detected by the device.
