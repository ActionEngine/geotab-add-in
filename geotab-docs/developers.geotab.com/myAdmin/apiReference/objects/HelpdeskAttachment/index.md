**Introduction**

This object describes the Attachment for the comment of an Helpdesk Ticket

**Properties**

## AttachmentSize

Attachment size, not required when POST attachment

## Body

byte array of the actual binary file

## CommentId

Comment Id of the ticket

## ContentType

Http Content Type Code

## CreatedAt

Attachment Uploaded Datetime

## Id

Attachment Object Id

## Name

File Name

## TicketId

Ticket Number of the ticket
