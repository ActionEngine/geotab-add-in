**Introduction**

Contains details of an RMA.

**Properties**

## Account

The[ApiAccount](../ApiAccount/index.md)that the RMA belongs to.

## CaseId

The ID of the support case related to the RMA.

## Comments

Comments related to the RMA.

## IssueDateUtc

Date the RMA was issued.

## ItemsCompletedExternal

Number of items completed.

## ItemsExpired

Number of items pending expired.

## ItemsPendingCredit

Number of items pending credit.

## ItemsPendingProcessed

Number of items pending processed.

## ItemsPendingReceiptExternal

Number of items pending receipt.

## ItemsPendingReplaced

Number of items pending replaced.

## ItemsPendingReturn

Number of items pending return.

## ItemsRejected

Number of items pending rejected.

## ItemsReplaced

Number of items replaced.

## OrderCount

Number of orders in the RMA.

## ResellerReference

Reseller's reference code to the RMA.

## RmaId

ID of the RMA.

## RmaType

The[ApiRmaType](../ApiRmaType/index.md)of the RMA.

## UserContact

The[ApiUserContact](../ApiUserContact/index.md)for the RMA.
