**Introduction**

Your public User account, including your UserId which you use for your Api Key. Extends[ApiNameEntity](../ApiNameEntity/index.md).

**Properties**

## AccountGroups

A list of the[ApiAccountIdGroupMapping](../ApiAccountIdGroupMapping/index.md)assigned to this user.

## Accounts

A list of the[ApiAccount](../ApiAccount/index.md)assigned to this user.

## FirstName

First Name of the user

## LastLogonDate

User last logon date

## LastName

Last Name of the user

## PhoneNumber

Phone number of the user

## Roles

A list of the[ApiRole](../ApiRole/index.md)assigned to this user.

## SessionId

The current Session ID

## UserId

Your active API Key
