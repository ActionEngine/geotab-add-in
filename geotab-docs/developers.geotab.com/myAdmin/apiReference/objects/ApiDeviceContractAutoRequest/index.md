**Introduction**

Contains device contract request information.

**Properties**

## Active

Indicates whether the request is still active (false means it is complete).

## AppliedDeviceContract

The[ApiDeviceContract](../ApiDeviceContract/index.md)that the request belongs to.

## Comments

Comments regarding the reason a contract was terminated

## Device

The[ApiGeotabDevice](../ApiGeotabDevice/index.md)that belongs to this request.

## DevicePlan

The[ApiDevicePlan](../ApiDevicePlan/index.md)that the device uses.

## Error

Any errors that occurred while processing the request.

## IsRental

Indicates whether the device is rented.

## Notified

Indicates whether a notification was sent to indicate that the request was processed.

## ProcessDate

Date the request was processed.

## RateCode

The rate code for this contract.

## Reason

Indicates the reason a contract was terminated (see[ApiDeviceContractRequestReason](../ApiDeviceContractRequestReason/index.md)).

## RequestDate

Date of the request.

## RequestInfo

Information about the type of request.

## RequestUser

The[ApiUser](../ApiUser/index.md)who made the request.

## Status

The status of the request.
