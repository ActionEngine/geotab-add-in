**Introduction**

Stores information about a Geotab device.

**Properties**

## DeviceType

Type of device ([ApiDeviceType](https://developers.geotab.com/myAdmin/apiReference/objects/ApiDeviceType/)).

## Id

Database Id of the device.

## ModemSerialNo

Serial number of the device IMEI or MEID

## SerialNumber

Serial number of the device.
