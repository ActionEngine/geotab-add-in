**Introduction**

Contains order status information.

**Properties**

## Account

Account where order was placed, used for billing.

## ActiveCurrencyRate

The[ApiCurrencyRate](../ApiCurrencyRate/index.md)active exchange rate for this order.

## Comments

Comments added by the reseller when the order was placed.

## CurrentOrderStatus

Current status of the order.

## EstimatedCompletionDate

Estimated shipping date.

## OnHold

Indicates whether the order is on hold.

## OnlineOrderItems

List of[ApiOnlineOrderItem](../ApiOnlineOrderItem/index.md)which corresponds to items in the order.

## OrderDate

Date order was placed.

## OrderedBy

Email address of the user who submitted the order.

## OrderMonthlyTotalLocal

Total monthly costs in the currency of[ActiveCurrencyRate](https://developers.geotab.com/myAdmin/apiReference/objects/ActiveCurrencyRate/).

## OrderNo

Geotab's order number.

## OrderSource

Order Source

## OrderTotalLocal

Total value of order in the currency of[ActiveCurrencyRate](https://developers.geotab.com/myAdmin/apiReference/objects/ActiveCurrencyRate/).

## PromoCode

Promo code that was used when order was placed.

## PurchaseOrderNo

Purchase order number for this order.

## RateCode

Rate code that was used when order was placed.

## ResellerReference

Reseller reference number provided when order was placed.

## ShipItems

List of[ApiOrderShipItem](../ApiOrderShipItem/index.md)corresponding to shipments that have been sent.

## ShippingContact

The[ApiUserContact](../ApiUserContact/index.md)that the order will be shipped to.

## ShippingCost

Shipping cost of the order.
