**Introduction**

This is the base object for MyAdmin Helpdesk Ticket

**Properties**

## AccountNumber

The account number that the ticket belongs to

## AdditionalDetails

Additional details, optional

## ApiResponse

Api Response field of the ticket, optional

## AssigneeName

Ticket Assignee Name

## Category

Ticket category

## CcEmail

carbon copy of the ticket, optional

## CodeSnippet

Code snippet field of the ticket, optional

## CommentList

Collection of the Comment Id, not required when POST a new ticket

## Context

Context field of the ticket, optional

## CreatedAt

Creation datetime of the ticket

## CreatedByEmail

Email Address of the ticket creator

## CreatedByName

Name of the ticket creator

## DatabaseName

MyGeotab Database Name, optional

## Description

The description of the ticket

## DeviceLights

Status lights on the GO Device, optional

## HowIssueBegin

How issue begins field of the ticket, optional

## HowToReproduce

How to Reproduce field of the ticket, optional

## Id

Ticket Number

## IsEvTicket

The ticket is related to EV, default false

## IsReadOnly

If true, no changes or replies are allowed to the ticket (Archived)

## IssueDescription

Issue Description field of the ticket, optional

## IssueEnvironment

Issue Environment field of the ticket, optional

## IssueIntermittent

Issue Intermittent field of the ticket, optional

## LastUpdatedAt

Last updated datetime of the ticket

## PoNumber

PO Number of the order, optional

## Priority

Ticket Priority, default is Normal

## Question

Question field of the ticket, optional

## SerialNumberAndDescription

String Dictionary of Go Device Serial Number and Description, optional

## Status

Status of the ticket

## Subcategory

Ticket subcategory

## Subject

The subject of the ticket

## TroubleshootingStepTaken

Troubleshoot step taken field of the ticket, optional

## Type

Ticket type

## WrapperLibraryUsed

Warpper Library used field of the ticket, optional
