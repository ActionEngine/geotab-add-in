**Introduction**

Contains status information about an item in an order.

**Properties**

## DevicePlan

The[ApiDevicePlan](../ApiDevicePlan/index.md)selected for devices in this order.

## ItemStatus

Status of the item in the order.

## MonthlyCost

Monthly cost per item.

## ProductCode

Product code for this item.

## ProductDescription

Description of the product.

## Quantity

Quantity ordered.

## UnitCost

Cost per item.
