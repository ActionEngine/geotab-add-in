**Introduction**

Contains the reason that a contract was terminated. This class extends[ApiNameEntity](../ApiNameEntity/index.md)- the "Name" property contains the reason description.

**Properties**

## DevicePlanId

ID of the[ApiDevicePlan](../ApiDevicePlan/index.md).

## Id

ID of the record.

## ParentId

ID of this[ApiDeviceContractRequestReason](index.md)'s parent[ApiDeviceContractRequestReason](index.md), if it has one. NULL if this is a top-level reason, with no parent.

## SortId

Sort order for[ApiDeviceContractRequestReason](index.md).
