**Introduction**

This is the helpdesk comment object that describing each comment of the Helpdesk Ticket

**Properties**

## AttachmentList

Collection of the attachment object Id, only populated when HasAttachment=True

## Comment

The message of the comment

## CreatedAt

The Comment Creation Datetime

## EmailAddress

The email address of the comment creator

## Id

Comment Id of the ticket, not required when POST a new comment

## Name

The name of the comment creator

## TicketId

Ticket number of the ticket

## Type

Type of the comment
