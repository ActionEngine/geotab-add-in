**Introduction**

A log record class used to add log record data for third party devices.

**Properties**

## DateTime

The[DateTime](https://developers.geotab.com/myAdmin/apiReference/objects/DateTime/)that the record was created.

## IsAuxiliary1On

True if auxiliary 1 was triggered, false otherwise.

## IsAuxiliary2On

True if auxiliary 2 was triggered, false otherwise.

## IsAuxiliary3On

True if auxiliary 3 was triggered, false otherwise.

## IsAuxiliary4On

True if auxiliary 4 was triggered, false otherwise.

## IsAuxiliary5On

True if auxiliary 5 was triggered, false otherwise.

## IsAuxiliary6On

True if auxiliary 6 was triggered, false otherwise.

## IsAuxiliary7On

True if auxiliary 7 was triggered, false otherwise.

## IsAuxiliary8On

True if auxiliary 8 was triggered, false otherwise.

## IsGpsValid

True if the device marked the GPS as valid, false otherwise.

## IsIgnitionOn

True if ignition was on, false otherwise.

## Latitude

The GPS latitude.

## Longitude

The GPS longitude.

## SerialNo

The device serial number.

## Speed

The GPS speed.
