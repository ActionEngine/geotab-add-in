**Introduction**

Stores header information for an order.

**Properties**

## AssignToId

ID of the[ApiUserContact](../ApiUserContact/index.md)to assign the devices to.

## Comments

Comments related to the order.

## ContractHeaderId

Contract Header ID. Only taken into consideration if PostOrderApiContractId feature flag is enabled

## CustomerId

ID of the[ApiCustomer](../ApiCustomer/index.md)to assign the devices to.

## DevicePlanLevel

The device plan level for this order.

## EmailCCList

Array of Email recipients to be sent with order. These email recipients will be added to notification process.

## ForAccount

Account for which order is being made.

## InternalComments

Comments to be sent to Geotab. These comments will not be shared with your customers.

## LeadId

Lead ID of the referrer for this order.

## LeadIdZip

Zip code corresponding to the Lead ID.

## OrderItems

Array of items ([ApiOrderItem](../ApiOrderItem/index.md)) in the order

## PromoCode

Promotion code for this order.

## PurchaseOrderNo

Purchase order number for the order.

## PxHubOpportunityId

PX Hub Opportunity ID

## RateCode

Rate plan code for this order.

## RequestedDeliveryDate

Requested delivery date.

## ResellerReference

Reseller reference information.

## RmaNumber

Related RMA number if the order is a replacement order.

## ShippingFeeId

The ID of the[ApiShippingFee](../ApiShippingFee/index.md)for this order.

## ShipToId

ID of the[ApiUserContact](../ApiUserContact/index.md)to ship the order to.
