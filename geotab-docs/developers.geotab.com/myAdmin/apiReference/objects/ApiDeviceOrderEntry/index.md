**Introduction**

Contains order status information.

**Properties**

## Account

Account where order was placed.

## Comments

Comments added by the reseller when the order was placed.

## DateTimeUtc

Date Shipped.

## Device

The[ApiGeotabDevice](../ApiGeotabDevice/index.md)device in this order.

## DeviceMeta

Populated if includeCarrierInfo is true

## DevicePlanLevel

Level of the device plan.

## PurchaseOrderNo

Purchase order number for this order.

## RateCode

Promo code that was used when order was placed.
