**Introduction**

Contains a database GUID and whether it is the owner or not for a specific device.

**Properties**

## Guid

Database Id of the device database.

## HardwareId

The id of the device assigned to the device.

## Name

The name of the database the device was assigned to.

## SerialNumber

The serial number of the device assigned to the device.

## VehicleClientType

The name of the database the device was assigned to.
