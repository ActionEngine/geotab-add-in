**Introduction**

Contains information related to a device's connection information.

**Properties**

## DeviceEnabled

The Devices' Enabled Status.

## DisabledParameters

The Devices' Disabled Parameters.

## EnabledParameters

The Devices' Enabled Parameters.

## FirmwareVersion

The Devices' Firmware Version.

## HardwareId

Hardware ID of the Device.

## LastConnect

The Devices' Last Connected Timestamp.

## LastEngine

The Devices' Last Engine Record Timestamp.

## LastGps

The Devices' Last GPS Record Timestamp.
