**Introduction**

Create a Helpdesk Attachment Object for a ticket

**Parameters**

## apiKey

The active API key used to authenticate the request.

## attachment

The[HelpdeskAttachment](../../objects/HelpdeskAttachment/index.md)object representing the content of the new attachment needed to be added to the ticket

## sessionId

The active session ID.

## ticketId

The requested ticket ID, must be under the requested account
