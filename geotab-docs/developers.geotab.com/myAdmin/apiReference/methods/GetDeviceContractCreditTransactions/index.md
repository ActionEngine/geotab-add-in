**Introduction**

Returns an array of[ApiDeviceContractCreditTransaction](../../objects/ApiDeviceContractCreditTransaction/index.md)objects for the given period and filters.

**Parameters**

## apiKey

The active API Key.

## forAccount

>Account to retrieve billing records for.

## monthFilter

The month number for the billing period (1 = January, 12 = December)

## sessionId

The active session ID.

## yearFilter

The 4-digit year for the billing period

**Return value**

Array of[ApiDeviceContractCreditTransaction](../../objects/ApiDeviceContractCreditTransaction/index.md)objects for the given period and filters
