**Introduction**

Returns an array of active[ApiDevicePlan](../../objects/ApiDevicePlan/index.md), available to all assigned accounts.

**Parameters**

## apiKey

The active API Key.

## forAccount

Limits results to device plans available to the specified account.

## sessionId

The active session ID.

**Return value**

Array of[ApiDevicePlan](../../objects/ApiDevicePlan/index.md).
