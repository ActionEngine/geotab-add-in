**Introduction**

API to bulk update ForceChangePassword for users

**Parameters**

## apiKey

The active API Key.

## forceChangePassword

Boolean to represent Force Change Password value

## sessionId

The active session ID.

## userNames

String array of user email address.

**Return value**

The[ApiForceChangePassword](../../objects/ApiForceChangePassword/index.md)object that contains list of updated users and invalid users.
