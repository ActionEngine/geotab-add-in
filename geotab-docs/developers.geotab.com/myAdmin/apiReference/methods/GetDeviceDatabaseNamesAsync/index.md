**Introduction**

Returns an array of[ApiDeviceDatabaseOwnerShared](../../objects/ApiDeviceDatabaseOwnerShared/index.md)containing devices and their owner databases and shared databases (if available).

**Parameters**

## apiKey

The active API Key.

## serialNumbers

The array of serial numbers for which to retrieve that list of owner and shared databases.

## sessionId

The active session ID.

**Return value**

Array of[ApiDeviceDatabaseOwnerShared](../../objects/ApiDeviceDatabaseOwnerShared/index.md).
