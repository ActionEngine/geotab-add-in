**Introduction**

Returns an array of[ApiSharedDatabase](../../objects/ApiSharedDatabase/index.md)containing databases the device has been assigned to and whether they are the owner or a shared database.

**Parameters**

## apiKey

The active API Key.

## serialNo

The device serial number to look up databases for.

## sessionId

The active session ID.

**Return value**

Array of[ApiSharedDatabase](../../objects/ApiSharedDatabase/index.md).
