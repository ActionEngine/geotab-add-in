**Introduction**

Gets an array of active[ApiRmaRequest](../../objects/ApiRmaRequest/index.md)for the specified account.

**Parameters**

## apiKey

The active API Key.

## forAccount

The account to retrieve RMA requests for.

## sessionId

The active session ID.

**Return value**

Array of[ApiRmaRequest](../../objects/ApiRmaRequest/index.md)of active RMA requests.
