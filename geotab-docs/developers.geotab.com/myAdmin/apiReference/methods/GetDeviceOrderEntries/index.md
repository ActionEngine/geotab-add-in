**Introduction**

Returns an array of[ApiDeviceOrderEntry](../../objects/ApiDeviceOrderEntry/index.md)containing status of orders.

**Parameters**

## apiKey

The active API Key.

## forAccount

Account where the order was created.

## includeCarrierInfo

If specified, include carrier info.

## poNum

Filter by purchase order number.

## sessionId

The active session ID.

**Return value**

Array of[ApiDeviceOrderEntry](../../objects/ApiDeviceOrderEntry/index.md).
