**Introduction**

Returns an array of[ApiDeviceJurisdiction](../../objects/ApiDeviceJurisdiction/index.md)containing devices and their jurisdiction and dig url.

**Parameters**

## apiKey

The active API Key.

## serialNumbers

The array of serial numbers for which to retrieve jurisdiction.

## sessionId

The active session ID.

**Return value**

Array of[ApiDeviceJurisdiction](../../objects/ApiDeviceJurisdiction/index.md).
