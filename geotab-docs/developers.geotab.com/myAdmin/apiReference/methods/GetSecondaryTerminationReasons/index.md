**Introduction**

Returns an array of[ApiDeviceContractRequestReason](../../objects/ApiDeviceContractRequestReason/index.md)that are possible reasons for termination.

**Parameters**

## apiKey

The active Api Key.

## primaryRequestReasonId

The primary termination request reason id for which to find secondary termination requests reasons.

## sessionId

The active session ID.

**Return value**

Array of[ApiDeviceContractRequestReason](../../objects/ApiDeviceContractRequestReason/index.md).
