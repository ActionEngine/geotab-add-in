**Introduction**

Returns current information relating to the device serial number provided.

**Parameters**

## apiKey

The active API Key.

## serialNo

A serial number of a Geotab GO device.

## sessionId

The active session ID.

**Return value**

Returns an[ApiDeviceInstallResult](../../objects/ApiDeviceInstallResult/index.md)object that contains current, relevant information pertaining to the device.
