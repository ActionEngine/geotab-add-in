**Introduction**

Gets system time in UTC (Coordinated Universal Time).

**Parameters**

## credentials

The user's Geotab login[Credentials](../../objects/Credentials/index.md).

**Return value**

The system DateTime when the call was processed on the server. This is in UTC format.
