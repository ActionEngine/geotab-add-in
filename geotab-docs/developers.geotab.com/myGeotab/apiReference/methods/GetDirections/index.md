**Introduction**

Gets step-by-step driving[Directions](../../objects/Directions/index.md)for a sequence of[Waypoint](../../objects/Waypoint/index.md)s including estimate travel time and distances.

**Parameters**

## credentials

The user's Geotab login[Credentials](../../objects/Credentials/index.md).

## pillar

The functional pillar for usage tracking.

## waypoints

The set of[Waypoint](../../objects/Waypoint/index.md)s.

**Return value**

Step-by-step[Directions](../../objects/Directions/index.md)between the[Waypoint](../../objects/Waypoint/index.md)s.
