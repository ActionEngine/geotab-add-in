**Introduction**

Upload a file for the corresponding[MediaFile](../../objects/MediaFile/index.md)using multipart/form-data POST request.

**Parameters**

## credentials

The user's Geotab login[Credentials](../../objects/Credentials/index.md).

## mediaFile

The[MediaFile](../../objects/MediaFile/index.md)of which to add the file for.

**Return value**

A[Task](https://docs.microsoft.com/en-us/dotnet/api/system.threading.tasks.task).
