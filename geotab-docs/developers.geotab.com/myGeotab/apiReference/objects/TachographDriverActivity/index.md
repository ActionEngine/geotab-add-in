**Introduction**

The entity that contains the data for the[TachographDriverActivity](index.md)tachograph driver activity extractor.

**Properties**

## Activity

The activity in this telemetry.

## DateTime

The time of this telemetry.

## Device

The[Device](../Device/index.md)id.

## Id

The unique identifier for the specific[Entity](../Entity/index.md)object in the Geotab system. See[Id](../Id/index.md).

## IsCrew

A value indicating if is part of the crew.

## Nation

The nation (specific country code from tachograph driver card).

## Slot

The slot.
