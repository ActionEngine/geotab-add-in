**Introduction**

The object used to specify the arguments when searching for[IoxAddOnStatus](../IoxAddOnStatus/index.md)(s).

**Properties**

## DeviceConnectionStatus

The device connection status (Active = 0, Inactive = 4, Disconnected = 1).

## DeviceSearch

Search for Iox Add-On Statusess that are or have been attached to this[DeviceSearch](../DeviceSearch/index.md) Id. Available DeviceSearch options are:.
- Id

## FromDate

Search for IoxAddOnStatus records that were logged at this date or after.

## Id

Search for an entry based on the specific[Id](../Id/index.md).

## ToDate

Search for IoxAddOnStatus records that were logged at this date or before.
