**Introduction**

The HOS log generation options.

**Properties**

## AUTO

If set to AUTO,[DutyStatusLog](../DutyStatusLog/index.md)will be automatically generated when the system determines the vehicle is using HOS features.

## OFF

HOS is disabled and generation of[DutyStatusLog](../DutyStatusLog/index.md)is disabled.

## ON

HOS is enabled and generation of[DutyStatusLog](../DutyStatusLog/index.md)is enabled.
