**Introduction**

The object used to specify the arguments when searching for[DutyStatusAvailability](../DutyStatusAvailability/index.md)objects.

**Properties**

## Id

Search for an entry based on the specific[Id](../Id/index.md).

## UserSearch

Search for duty status availability objects with this[UserSearch](../UserSearch/index.md). Available UserSearch options are:.
- Id
- DriverGroups
- DriverGroupFilterCondition
