**Introduction**

The object used to specify the arguments when searching for[Trailer](../Trailer/index.md)(s).

**Properties**

## Groups

Search for Trailers that are members of these[GroupSearch](../GroupSearch/index.md)(s) one of it's children or one of it's parents. Available GroupSearch options are:.
- Id

## Id

Search for an entry based on the specific[Id](../Id/index.md).

## Name

Search for Trailers with a name containing this[String](https://docs.microsoft.com/en-us/dotnet/api/system.string). Wildcard can be used by prepending/appending "%" to string.
