**Introduction**

The object used to specify the arguments when searching for[Controller](../Controller/index.md)(s).

**Properties**

## Code

Search for a Controller by the code number.

## Id

Search for an entry based on the specific[Id](../Id/index.md).

## Ids

Search for Controllers with these unique[Id](../Id/index.md)(s).

## Name

Search for Controllers with this Name. Wildcard can be used by prepending/appending "%" to string. Example "%name%".

## SourceSearch

Search for Controllers with this[SourceSearch](../SourceSearch/index.md) Id. Available SourceSearch options are:.
- Id
