**Introduction**

The object used to specify the arguments when searching for[AnnotationLog](../AnnotationLog/index.md)(s).

**Properties**

## Comment

Search for AnnotationLogs which have a comment that contains this[String](https://docs.microsoft.com/en-us/dotnet/api/system.string). Wildcard can be used by prepending/appending "%" to string.

## FromDate

Search for AnnotationLogs that were recorded at this date or after.

## Id

Search for an entry based on the specific[Id](../Id/index.md).

## ToDate

Search for AnnotationLogs that were recorded at this date or before.

## UserSearch

Search for AnnotationLogs with this[UserSearch](../UserSearch/index.md) Id. Available UserSearch options are:.
- Id
