**Introduction**

The object used to specify the arguments when searching for[ShipmentLog](../ShipmentLog/index.md)(s).

**Properties**

## Commodity

Search for ShipmentLogs with this commodity. Wildcard can be used by prepending/appending "%" to string.

## DeviceSearch

Search for ShipmentLogs for this[DeviceSearch](../DeviceSearch/index.md) Id. Available DeviceSearch options are:.
- Id

## DocumentNumber

Search for ShipmentLogs with this document number. Wildcard can be used by prepending/appending "%" to string.

## FromDate

Search for ShipmentLogs that were active at this date or after. Set to UTC now to search for only currently active ShipmentLogs.

## Id

Search for an entry based on the specific[Id](../Id/index.md).

## Keywords

Search for entities that contain specific keywords in all wildcard string-searchable fields.

## ShipperName

Search for ShipmentLogs with this shipper name. Wildcard can be used by prepending/appending "%" to string.

## ToDate

Search for ShipmentLogs that were active at this date or before.

## UserSearch

Search for ShipmentLogs with this[UserSearch](../UserSearch/index.md) Id. Available UserSearch options are:.
- Id
