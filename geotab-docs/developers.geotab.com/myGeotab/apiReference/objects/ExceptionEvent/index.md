**Introduction**

The event of an exception generated by Rule violation.

**Properties**

## ActiveFrom

The start date of the ExceptionEvent; at or after this date.

## ActiveTo

The end date of the ExceptionEvent; at or before this date.

## Device

The[Device](../Device/index.md)specified for the device.

## Distance

The km distance traveled since the start of the ExceptionEvent.

## Driver

The[Driver](../Driver/index.md)specified for the device.

## Duration

The duration of the violation.

## ExceptionCount

The count of exception instances for this exception event. This property is only populated when[IncludeExceptionCount](https://developers.geotab.com/myGeotab/apiReference/objects/IncludeExceptionCount/)is set to true.

## Id

The unique identifier for the specific[Entity](../Entity/index.md)object in the Geotab system. See[Id](../Id/index.md).

## LastModifiedDateTime

The time this exception event was last updated. Default [MinDate].

## Rule

The[Rule](../Rule/index.md)which was violated.

## State

The[ExceptionEventState](../ExceptionEventState/index.md)of the exception.

## Version

The version of the entity.
