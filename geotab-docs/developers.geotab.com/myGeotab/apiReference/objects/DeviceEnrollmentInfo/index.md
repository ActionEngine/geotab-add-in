**Introduction**

Represents the info of a device to be enrolled for Clean Truck Check emission reporting.

**Properties**

## Device

The device to be enrolled. Available options to identify the device are:
- Id
- SerialNumber

## DmvRegistrationDateTime

The DMV registration datetime.

## LoggingFrequencyType

The[EmissionEnrollmentLoggingFrequencyType](../EmissionEnrollmentLoggingFrequencyType/index.md).

## VehicleIdentificationNumber

The vehicle identification number if used to identify the device.
