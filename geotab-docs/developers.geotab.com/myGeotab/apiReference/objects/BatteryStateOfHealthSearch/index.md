**Introduction**

The object used to specify the arguments when searching for a[BatteryStateOfHealth](../BatteryStateOfHealth/index.md).

**Properties**

## DeviceSearch

Filter by the[DeviceSearch](../DeviceSearch/index.md)options. Providing a device ID will search for any BatteryStateOfHealth(s) that are assigned to that Device. Providing deviceIds will search for any BatteryStateOfHealth(s) that are assigned to these Devices. Providing the Groups will search for BatteryStateOfHealth(s) for that have Devices in that group. Available DeviceSearch options are:
- Id
- DeviceIds
- Groups

## FromDate

The from date, which is used to search for BatteryStateOfHealth(s) calculated on or after this date.

## Id

Search for an entry based on the specific[Id](../Id/index.md).

## ToDate

The to date, which is used to search for BatteryStateOfHealth(s) calculated on or before this date.
