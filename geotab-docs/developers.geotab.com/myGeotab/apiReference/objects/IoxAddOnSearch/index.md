**Introduction**

The object used to specify the arguments when searching for[IoxAddOn](../IoxAddOn/index.md)(s).

**Properties**

## ConnectedOnly

If <see langword="true" only Iox Add-Ons that are currently connected to a device will be returned.

## DeviceSearch

Search for Iox Add-Ons that are or have been attached to this[DeviceSearch](../DeviceSearch/index.md) Id. Available DeviceSearch options are:.
- Id

## Id

Search for an entry based on the specific[Id](../Id/index.md).

## Type

Search for Iox Add-Ons of a specific type (GoTalk, NFC, etc.). See[KnownIoxAddOnType](../KnownIoxAddOnType/index.md).
