**Introduction**

Represents the data flow type for a[DeviceShare](../DeviceShare/index.md).

**Properties**

## Incoming

Indicates that an active[DeviceShare](../DeviceShare/index.md)means that device data from a different database is being shared with this one.

## Outgoing

Indicates that an active[DeviceShare](../DeviceShare/index.md)means that device data from this database is being shared with a different one.
