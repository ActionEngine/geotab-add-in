**Introduction**

An DefectRemark is a remark that can be associated with a[DVIRDefect](../DVIRDefect/index.md).

**Properties**

## DateTime

The date and time the log was created.

## Id

The unique identifier for the specific[Entity](../Entity/index.md)object in the Geotab system. See[Id](../Id/index.md).

## Remark

The DVIR text associated with the log.

## User

The[User](../User/index.md)who created the log.
