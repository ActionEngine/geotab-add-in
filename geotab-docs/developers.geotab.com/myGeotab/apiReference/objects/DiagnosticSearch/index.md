**Introduction**

The object used to specify the arguments when searching for[Diagnostic](../Diagnostic/index.md)(s).

**Properties**

## Code

Search for a Diagnostic by the code number.

## DiagnosticType

The[DiagnosticType](../DiagnosticType/index.md)to search for in Diagnostics.

## DiagnosticTypes

The list of[DiagnosticType](../DiagnosticType/index.md)to search for in Diagnostics.

## Id

Search for an entry based on the specific[Id](../Id/index.md).

## Ids

Search for Diagnostics with these unique[Id](../Id/index.md)(s).

## Name

Search for Diagnostics with this Name. Wildcard can be used by prepending/appending "%" to string. Example "%name%".

## SourceSearch

The[SourceSearch](../SourceSearch/index.md) Id to search for in Diagnostics. Available SourceSearch options are:.
- Id
- Name
