**Introduction**

A unique identifier for different types[IoxAddOn](../IoxAddOn/index.md)s. The range of valid values is between 4096 (inclusive) and 8192.

**Properties**

## Bluetooth

The Bluetooth Iox Add-On type. Integer value is 4134.

## GoTalk

The GoTalk Iox Add-On type. Integer value is 4125.

## GoTalkV2

The GoTalk V2 Iox Add-On type. Integer value is 4242.

## IoxBridge

The IOX-BRIDGE Add-On type. Integer value is 4241.

## IoxOutput

The Output Iox Add-On type. Integer value is 4135.

## Iridium

The Iridium Iox Add-On type. Integer value is 4097.

## NFC

The NFC Iox Add-On type. Integer value is 4110.

## UReader

The U Reader Iox Add-On type. Integer value is 4168.
