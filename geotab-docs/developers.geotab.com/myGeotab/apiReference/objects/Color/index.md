**Introduction**

Specifies the color to use to identify the[Group](../Group/index.md),[Rule](../Rule/index.md),[SecurityClearance](../SecurityClearance/index.md)or[Zone](../Zone/index.md).

**Properties**

## A

Returns the alpha component of the color space (opacity) in the range 0-255.

## B

Returns the blue component of the color space in the range 0-255 from the default sRGB space.

## G

Returns the green component of the color space in the range 0-255 from the default sRGB. space.

## R

Returns the red component of the color space in the range 0-255 from the default sRGB. space.
