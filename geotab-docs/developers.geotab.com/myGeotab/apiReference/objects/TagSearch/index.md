**Introduction**

The object used to specify the arguments when searching[Tag](../Tag/index.md)entries.

**Properties**

## Id

Search for an entry based on the specific[Id](../Id/index.md).

## Name

Search for Tags with this Name. Wildcard can be used by prepending/appending "%" to string. Example "%name%".

## Names

Search for tag names. Supports up to 500 names. Does not support wildcards.

## TagIds

Search for Tag with given Tag Ids. Supports up to 500 Ids.
