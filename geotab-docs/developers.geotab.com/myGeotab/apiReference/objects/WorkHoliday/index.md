**Introduction**

Day that is specified as not being a regular working day.

**Properties**

## Date

The midnight of the work holiday.

## HolidayGroup

The holiday group for this holiday day. See[WorkTimeHolidayGroupId](../WorkTimeHolidayGroupId/index.md).

## Id

The unique identifier for the specific[Entity](../Entity/index.md)object in the Geotab system. See[Id](../Id/index.md).

## Name

The name of this entity which identifies it and is used when displaying this entity.
