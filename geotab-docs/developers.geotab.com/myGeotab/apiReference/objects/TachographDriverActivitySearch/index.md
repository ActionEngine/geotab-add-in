**Introduction**

The object used to specify the arguments when searching for[TachographDriverActivity](../TachographDriverActivity/index.md).

**Properties**

## Extrapolate

A value indicating whether if we want to use or not extrapolation in the telemetry.

## FromDate

The initial date range (i.e. 2022-04-01T00:00:00.000Z).

## Id

Search for an entry based on the specific[Id](../Id/index.md).

## ToDate

The final date range (i.e. 2022-04-03T00:00:00.000Z).

## Type

The telemetry (file, stream, combine).
