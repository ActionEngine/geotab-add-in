**Introduction**

Message content that can send a GPS location to a device. Derived from[TextContent](../TextContent/index.md).

**Properties**

## Address

The address of the stop.

## ContentType

The type of message. See[MessageContentType](../MessageContentType/index.md).

## Id

The[Id](../Id/index.md)of the route message.

## Latitude

The latitude of the stop.

## Longitude

The longitude of the stop.

## Message

Contains the message text.

## OnRouteRanges

A collection of "DateRange"s indicating when the driver's intent was to go to the provided[Latitude](https://developers.geotab.com/myGeotab/apiReference/objects/Latitude/)and[Longitude](https://developers.geotab.com/myGeotab/apiReference/objects/Longitude/).

## RouteId

The[Id](../Id/index.md)of the route message.
