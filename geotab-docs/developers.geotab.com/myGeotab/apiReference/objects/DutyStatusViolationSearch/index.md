**Introduction**

The object used to specify the arguments when searching for[DutyStatusViolation](../DutyStatusViolation/index.md)(s). This search has been designed to work efficiently with these combinations of parameters:
- UserSearch + FromDate and/or ToDate

**Properties**

## FromDate

Search for duty status violations that were recorded at this date or after.

## Id

Search for an entry based on the specific[Id](../Id/index.md).

## ToDate

Search for duty status violations that were recorded at this date or before.

## UserSearch

Search for duty status violations with this[UserSearch](../UserSearch/index.md). Available UserSearch options are:.
- Id
- CompanyGroups
- DriverGroups
