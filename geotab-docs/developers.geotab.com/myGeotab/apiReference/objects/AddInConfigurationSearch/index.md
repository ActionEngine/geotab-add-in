**Introduction**

The object used to specify the arguments when searching for[AddInConfiguration](../AddInConfiguration/index.md).

**Properties**

## Id

Search for an entry based on the specific[Id](../Id/index.md).

## Name

The name of this Add-In.

## SolutionId

The solution id related to this Add-In.

## SupportEmail

The email address for support related to this Add-In.
