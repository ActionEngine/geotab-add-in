**Introduction**

WifiHotspot is used to configure WiFi hotspot settings on telematics devices.

**Rate limits**

| Method | Description | Limit | Period | Status |
| --- | --- | --- | --- | --- |

| Add | Limit of 60 Add requests per 1m. | 60 | 1m | Active |
| Get | Limit of 60 Get requests per 1m. | 60 | 1m | Active |
| Set | Limit of 60 Set requests per 1m. | 60 | 1m | Active |
| Remove | Limit of 60 Remove requests per 1m. | 60 | 1m | Active |
| GetCountOf | Limit of 60 GetCountOf requests per 1m. | 60 | 1m | Active |
