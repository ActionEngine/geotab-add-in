**Introduction**

The contents of a[TextMessage](../TextMessage/index.md)that will be delivered to a GoTalk.

**Properties**

## ContentType

The type of message. See[MessageContentType](../MessageContentType/index.md).

## ExpectedDuration

The expected round trip time in seconds of the message including time for the GoTalk to read the message.

## Message

The message. The message can be up to 256 characters.
