**Introduction**

The object used to specify the arguments when searching for a[WifiHotspot](../WifiHotspot/index.md).

**Properties**

## DeviceSearch

Filter by the[DeviceSearch](../DeviceSearch/index.md)options. Providing a device ID will search for any VehicleConfiguration that are assigned to that Device. Providing the Groups will search for VehicleConfiguration for that have Devices in that group. Available DeviceSearch options are:
- Id
- Groups

## Id

Search for an entry based on the specific[Id](../Id/index.md).
