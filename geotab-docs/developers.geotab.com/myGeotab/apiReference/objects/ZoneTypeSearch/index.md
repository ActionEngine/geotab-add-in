**Introduction**

The object used to specify the arguments when searching for[ZoneType](../ZoneType/index.md)(s).

**Properties**

## Id

Search for an entry based on the specific[Id](../Id/index.md).

## IsBuiltIn

Search zone types that are or are not built in.

## Keywords

Search zone types that contain specific keywords in name and comment fields.

## Name

Name of ZoneType to search for. Wildcard can be used by prepending/appending "%" to string. Example "%comments%".
