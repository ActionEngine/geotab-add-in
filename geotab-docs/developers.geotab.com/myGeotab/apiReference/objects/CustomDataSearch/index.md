**Introduction**

The object used to specify the arguments when searching for[CustomData](../CustomData/index.md).

**Properties**

## DeviceSearch

Search for CustomData recorded for this[DeviceSearch](../DeviceSearch/index.md) Id. Available DeviceSearch options are:.
- Id

## FromDate

Search for CustomData that was recorded at this date or after.

## Id

Search for an entry based on the specific[Id](../Id/index.md).

## ToDate

Search for CustomData that was recorded at this date or before.
