**Introduction**

The object used to specify the arguments when searching for a[FailureMode](../FailureMode/index.md).

**Properties**

## Code

Search for a FailureMode by the code number.

## Id

Search for an entry based on the specific[Id](../Id/index.md).

## Ids

Search for FailureMode with these unique[Id](../Id/index.md)(s).
