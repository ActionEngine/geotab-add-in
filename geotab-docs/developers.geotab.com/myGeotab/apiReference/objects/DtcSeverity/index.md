**Introduction**

Represents a severity/class code from the engine system of the specific[Device](../Device/index.md).

**Properties**

## CheckAtNextHalt

This value indicates to the failure that a check of the vehicle is required at next halt. Displayed in the UI with a "High" severity level.

## CheckImmediately

This value indicates to the failure that an immediate check of the vehicle is required. Displayed in the UI with a "Critical" severity level.

## MaintenanceOnly

This value indicates that the failure requests maintenance only. Displayed in the UI with a "Medium" severity level.

## NoRepairNeeded

This value indicates that no repair is needed. Displayed in the UI with a "Low" severity level.
