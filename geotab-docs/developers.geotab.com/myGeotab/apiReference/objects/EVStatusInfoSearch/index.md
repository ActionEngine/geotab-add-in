**Introduction**

The object used to specify the arguments when searching for a[EVStatusInfo](../EVStatusInfo/index.md).

**Properties**

## DeviceSearch

Filter by the[DeviceSearch](../DeviceSearch/index.md)options. Providing a device ID will search for any EVStatusInfo(s) that are assigned to that Device. Providing deviceIds will search for any EVStatusInfo(s) that are assigned to these Devices. Providing the Groups will search for EVStatusInfo(s) for that have Devices in that group. Available DeviceSearch options are:
- Id
- DeviceIds
- Groups

## Id

Search for an entry based on the specific[Id](../Id/index.md). Note: This property is not available in Beta.
