**Introduction**

A message that requests the current location of a device through Iridium.

**Properties**

## Device

The device associated with the location request Used to look up device in BinaryPayloadBridge

## Id

The unique identifier for the specific[Entity](../Entity/index.md)object in the Geotab system. See[Id](../Id/index.md).

## Payload

The raw binary data contained in the message.

## PayloadType

The[BinaryPayloadType](../BinaryPayloadType/index.md).
