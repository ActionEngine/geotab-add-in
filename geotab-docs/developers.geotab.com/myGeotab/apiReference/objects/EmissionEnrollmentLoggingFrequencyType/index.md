**Introduction**

Represents Clean Truck Check emission enrollment logging frequency types.

**Properties**

## Name

The name for the[EmissionEnrollmentLoggingFrequencyType](index.md).

## Value

The numeric value representing a logging frequency type. Valid values are:
- 0 (Disabled)
- 1~250 (Every 1~250 Day)
- 251 (Every Ignition)
- 253 (On Demand)
