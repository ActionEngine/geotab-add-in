**Introduction**

The type of[TextMessage](../TextMessage/index.md)content.

**Properties**

## CannedResponse

Text message that also includes response options. See[CannedResponseContent](../CannedResponseContent/index.md).

## ColdChainFaultClearContent

Content contains MimeContent to reset a Cold Chain fault code.

## ColdChainSetpointSetContent

Content contains MimeContent to set a setpoint temperature.

## DriverAuthList

Text message with information to add/remove a driver from a[GoDevice](../GoDevice/index.md)'s auth list. See[DriverAuthListContent](../DriverAuthListContent/index.md).

## GoTalk

Text message that is converted to speech. Must have GOTalk.

## IoxOutput

Text message that includes instructions to open or close an IOX-OUTPUT relay. See[IoxOutputContent](../IoxOutputContent/index.md).

## Location

Text message that includes a location. See[LocationContent](../LocationContent/index.md).

## MimeContent

Text message that contains bytes in a predefined format that describes the MimeType to a third-party IOX Add-On over an RS232 connection. See[MimeContent](../MimeContent/index.md).

## Normal

Basic text message. See[TextContent](../TextContent/index.md).
