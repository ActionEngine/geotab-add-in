**Introduction**

The object used to specify the arguments when searching[DebugData](../DebugData/index.md).

**Properties**

## DebugReason

The search for DebugData records with[DebugReason](https://developers.geotab.com/myGeotab/apiReference/objects/DebugReason/). Note that it's ignored by the GetFeed Search function.

## DeviceSearch

The search for DebugData recorded for this[DeviceSearch](../DeviceSearch/index.md) Id. Available DeviceSearch options are:.
- Id

## FromDate

The date after which DebugData was recorded.

## Id

Search for an entry based on the specific[Id](../Id/index.md).

## ToDate

The date before which DebugData was recorded.
