**Introduction**

An event representing fuel level of a vehicle. An extremum representing either the minimum or maximum point of fuel used for a given[FillUp](../FillUp/index.md).

**Properties**

## Data

The data of the extremum. This is derived from the[StatusData](../StatusData/index.md)record used to calculate the extremum.

## DateTime

The date time of the extremum. This is derived from the[StatusData](../StatusData/index.md)record used to calculate the extremum.

## Source

The extremum type as the[FuelTankCapacitySource](../FuelTankCapacitySource/index.md). This is derived from the specific diagnostic type of the[StatusData](../StatusData/index.md)records used to calculate the extremum.
